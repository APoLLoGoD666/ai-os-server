Risk & Architecture Notes OBSERVATIONS

server.js monolith (525KB)
All 130+ routes in a single file. High coupling — any syntax error halts the entire service. Difficult to test individual domains.
HIGH · Split by domain recommended
In-process concurrency (3-slot queue)
Agent queue is process-scoped. Render single-instance. Under high load, tasks queue or drop. No worker handoff.
MED · Queue externalization (Redis/BullMQ) for scale
_active_count governance (process-scoped)
Concurrency limits in governance wrapper are per-process. Multi-worker deployments would bypass SAFE_SINGLE_THREAD guarantees.
MED · Redis-backed counter for multi-worker
Session TTL (15 min in-memory)
Session registry is in-process. Render restarts clear all sessions. Users lose mid-conversation state.
MED · Externalise session store (Supabase KV)
Governance isolation (advisory)
runtime_constraints in sandbox context are passed to agent fn but agents CAN ignore them. True enforcement requires VM sandbox.
LOW · Acceptable for current trusted agent scope
Cost tracking without hard budget kill
PIPELINE_BUDGET_USD is tracked but enforcement is soft. A runaway agent loop could exceed cost cap before being caught.
LOW · Hard-kill at budget threshold recommended
Obsidian vault path dependency
obsidian-client.js falls back to local filesystem (OBSIDIAN_VAULT_PATH) on Render — not persistent across deploys without a bind mount.
INFO · Use Supabase Storage for vault persistence
Playwright on Render (memory)
Chromium is resource-heavy. Parallel research calls (/api/browser/research-parallel) can spike memory on the Render free/starter tier.
INFO · Monitor memory — upgrade tier if browsers used heavily
Architecture Strengths
✓Zero-throw contracts — all governance modules return error objects, never throw. Deterministic execution across the control plane.
✓Frozen object graph — all governance outputs are Object.freeze'd. No accidental mutation across async boundaries.
✓Type-agnostic registry — governance layer never hardcodes agent types. Extensible indefinitely via capabilities.
✓Coherence-gated execution — system health directly controls agent autonomy level. Degraded state = restricted execution automatically.
✓Layered memory ACL — single gateway, 11 layers, access-controlled per entity, PII-sanitized on every retrieval path.
✓Circuit breaker + reflection loop — failures auto-throttle API calls and feed lessons back into memory for future improvement.
